About me

I am an iCan student that started the program this year. In iCan we do assignments that require scripting, 3D modeling, Graphic Design, and many more skills but of all the skills that we need I like 3D modeling the most.

Template by TemplateMo

What is 3D modeling?/What softwares I use

3D modeling is the process of creating an object, a location, or an animal in 3D using 3D shapes like cubes, cylinders, and transforming them into complex shapes. I mainly use Blender, and occasionally Tinkercad.

What is Graphic Design?/What software I use

Graphic Design involves editing images, creating logos, and video effects. I use Adobe Photoshop, Illustrator, and After Effects.

What is Scripting?/What softwares I use

Scripting is creating code for games, automation, or websites. I use Zorin Terminal (Linux), Thonny, Raspberry Pi Editor, GitHub, and Windows Terminal.
