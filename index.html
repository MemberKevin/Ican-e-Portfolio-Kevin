What is 3D modeling?/What softwares I use

3D modeling is the process of creating on object, a location, or an animal in a 3D using 3D shapes like cubes, cylinders and transforming them into complex shapes using the many tools you have to work with. I use mainly Blender as my 3D modeling software of choice. Occasionally I use Tinkercad.

What is Graphic Design?/What software I use

Graphic Design is the act of photoshopping an image and adding your own twist to it or it can be creating logos for businesses/companies or creating effects for a music video. For graphic design I use Adobe Photoshop, Adobe Illustrator, and Adobe After Effects.

What is Scripting?/What softwares I use

Scripting is the process of creating code for games, cleaning up your PC's storage or creating websites. Scripting is a great way to learn about your PC's roots and branches to all of the files. To script I use Zoron's terminal (a Linux distibution), Thonny, Raspberry Pi editor, Github, And Windows terminal.
